## 🚀 What it does

agent-memory stores an AI agent’s memory in plain markdown files. It helps your agent keep track of facts, notes, tasks, and past work across sessions.

It uses files instead of a database, so it stays easy to inspect, edit, and move. You can open the memory files in any text editor.

## 📁 How it stores memory

agent-memory uses simple markdown files. That means:

- you can read the content in Notepad
- you can back it up with File Explorer
- you can move it to another PC
- you do not need a database
- you can keep each agent in its own folder

A common setup looks like this:

- `memory/`
  - `agent-name/`
    - `profile.md`
    - `facts.md`
    - `tasks.md`
    - `history.md`

## 🛠️ How to use it

1. Start the app
2. Choose or create an agent folder
3. Add notes, facts, or task items
4. Let the agent read and update the files
5. Open the markdown files any time to check what is stored

If you use more than one agent, give each one a separate folder. This keeps memory clean and easy to follow.
